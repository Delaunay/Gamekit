Source Code

# Go to your project directory
cd /path/to/my/uproject

# Add Gamekit as a submodule
git submodule add https://github.com/Delaunay/Gamekit.git Plugins/Gamekit

Binary Installation Procedure

  1. Unzip inside <Project>/Plugins

Project structure after unzipping

<Project>/
├── Config/
│   └── DefaultEditor.ini
├── Plugins/
|   └── Gamekit
|       ├── Source
|       └── Gamekit.uplugin
├── Shaders
├── Source
└── <Project>.uproject

  1. Make the project load the plugin by modifying <Project>.uproject

"Modules": [
        {
                "Name": "<Project>",
                "Type": "Runtime",
                "LoadingPhase": "Default",
                "AdditionalDependencies": [
                        "Engine",
                        "AIModule"
                ]
        },
        {
                "Name": "Gamekit",
                "Type": "Runtime",
                "LoadingPhase": "Default",
                "AdditionalDependencies": [
                        "Engine",
                        "AIModule"
                ]
        }
],

  1. Enable your C++ project to use Gamekit Source/<Project>/<Project>.Build.cs

PublicDependencyModuleNames.AddRange(new string[] {
        ...
        "Gamekit"
});
